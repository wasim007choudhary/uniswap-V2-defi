# Why Did Profit Start Falling?

Not because the arbitrage disappeared.

Not because the code changed.

Not because someone stole our money.

The answer is much simpler.

> **Our own trade changed the prices.**

By trading too much,

we destroyed the very opportunity we wanted to exploit.

This is one of the most important ideas in arbitrage.

Your own trade creates slippage.

Enough slippage eventually eliminates your own profit.

---

# The Big Realization

Notice something.

Our goal is **NOT**

```
Largest Trade
```

Our goal is

```
Largest Profit
```

These are **not** the same thing.

Many beginners accidentally confuse the two.

The biggest trade often produces **less** profit than a smaller one because of slippage.

---

# So What Do We Actually Want?

We don't want

```
The Biggest Trade
```

We don't want

```
All Our Money
```

We don't want

```
Maximum Amount In
```

Instead,

we want the trade size located exactly at

```
★
```

because

that trade size produces

```
Maximum Profit.
```

This trade size is called

# **Optimal Amount In**

Notice the wording.

Not

```
Maximum Amount In
```

Not

```
Largest Amount In
```

But

```
Optimal Amount In
```

Meaning

> **The exact amount of tokens we should trade to produce the highest possible profit.**

# But How Do We Find ★?

Imagine I ask you

```

```

```
1.25 DAI

...

...

1,000,000 DAI
```

Testing every possible value is impossible.

So mathematicians asked a different question.

> **"Can we calculate the best trade size directly without testing every possible amount?"**

The answer is

**Yes.**

And that is exactly why calculus is introduced in this lesson.

# Final Takeaway

This lesson is **not** about learning calculus.

It is **not** about memorizing formulas.

It is **not** about quadratic equations.

Those are simply the tools.

The real problem this lesson is solving is much simpler.

> **Given an arbitrage opportunity, how do we calculate the exact trade size that produces the maximum possible profit before slippage starts reducing it?**

Everything that follows—the derivative, the quadratic equation, and the formulas for `a`, `b`, `c`, and `k`—exists solely to answer that one question.
