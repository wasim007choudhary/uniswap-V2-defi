# Step F4 – Eliminating the Intermediate Variable Using the Fee Multiplier

> **Goal of this step**
>
> We have derived the fee-adjusted swap equations for both AMMs:
>
> **First Swap**
>
> \[
> dx_a=\frac{fx_ar}{y_a+fr}
> \]
>
> **Second Swap**
>
> \[
> dy_b=\frac{fy_bdx_a}{x_b+fdx_a}
> \]
>
> where
>
> \[
> f=\frac{997}{1000}
> \]
>
> and
>
> - \(r\) is the trader's initial input.
> - \(dx_a\) is the output of the first swap.
> - \(dy_b\) is the final output after the second swap.
>
> Our objective is to eliminate the intermediate variable \(dx_a\), expressing the entire arbitrage as a function of only the initial input \(r\).

---

# Why Eliminate \(dx_a\)?

Currently the arbitrage looks like

```
Initial Input (r)

↓

First Swap

↓

dxₐ

↓

Second Swap

↓

dy_b
```

Notice that

\[
dy_b
\]

still depends on

\[
dx_a.
\]

Later, we'll differentiate the profit function with respect to the **initial input**.

Therefore every intermediate variable must disappear.

Our desired dependency becomes

```
r

↓

dy_b
```

---

# Step 1 — Write Both Swap Equations

From the first swap,

\[
dx_a=\frac{fx_ar}{y_a+fr}
\]

From the second swap,

\[
dy_b=\frac{fy_bdx_a}{x_b+fdx_a}
\]

Everything now depends on substituting the first equation into the second.

---

# Step 2 — Substitute \(dx_a\)

Replace every occurrence of \(dx_a\).

The numerator becomes

\[
fy_b
\left(
\frac{fx_ar}{y_a+fr}
\right)
\]

The denominator becomes

\[
x_b+
f
\left(
\frac{fx_ar}{y_a+fr}
\right)
\]

Therefore,

\[
dy_b=
\frac{
fy_b
\left(
\frac{fx_ar}{y_a+fr}
\right)
}
{
x_b+
f
\left(
\frac{fx_ar}{y_a+fr}
\right)
}
\]

At first glance this looks intimidating.

However, it is simply a fraction whose denominator contains another fraction.

The strategy is exactly the same as in the zero-fee derivation.

---

# Step 3 — Simplify The Numerator

Multiply the constants.

Since

\[
f\times f=f^2,
\]

the numerator simplifies to

\[
\frac{f^2x_ay_br}
{y_a+fr}
\]

The equation now becomes

\[
dy_b=
\frac{
\dfrac{f^2x_ay_br}
{y_a+fr}
}
{
x_b+
\dfrac{f^2x_ar}
{y_a+fr}
}
\]

Notice something important.

Both the numerator and one term of the denominator share the same denominator,

\[
y_a+fr.
\]

This is exactly what will allow the expression to collapse.

---

# Step 4 — Give The Denominator A Common Denominator

The denominator currently is

\[
x_b+
\frac{f^2x_ar}
{y_a+fr}.
\]

Rewrite

\[
x_b
\]

using the common denominator

\[
y_a+fr.
\]

This gives

\[
x_b=
\frac{x_b(y_a+fr)}
{y_a+fr}.
\]

Now both terms have the same denominator.

Therefore,

\[
x_b+
\frac{f^2x_ar}
{y_a+fr}
=
\frac{
x_b(y_a+fr)+f^2x_ar
}
{y_a+fr}.
\]

Our equation becomes

\[
dy_b=
\frac{
\dfrac{f^2x_ay_br}
{y_a+fr}
}
{
\dfrac{
x_b(y_a+fr)+f^2x_ar
}
{y_a+fr}
}.
\]

---

# Step 5 — Divide By A Fraction

Recall the algebraic identity

\[
\frac{A}{B/C}
=
A\cdot\frac{C}{B}.
\]

Apply that rule.

\[
dy_b=
\frac{f^2x_ay_br}
{y_a+fr}
\cdot
\frac{y_a+fr}
{x_b(y_a+fr)+f^2x_ar}.
\]

Now notice something beautiful.

The factor

\[
y_a+fr
\]

appears in both places.

It cancels immediately.

---

# Step 6 — Final Simplified Equation

After cancellation,

\[
\boxed{
dy_b=
\frac{
f^2x_ay_br
}
{
x_b(y_a+fr)+f^2x_ar
}
}
\]

Expand the denominator.

\[
x_b(y_a+fr)
=
x_by_a+
fx_br.
\]

Therefore,

\[
\boxed{
dy_b=
\frac{
f^2x_ay_br
}
{
x_by_a+
(fx_b+f^2x_a)r
}
}
\]

This is the complete fee-adjusted arbitrage equation.

Notice that

- \(dx_a\) has disappeared.
- Every quantity depends only on the trader's initial input \(r\).

This is exactly what we need before differentiation.

---

# Compare With The Zero-Fee Result

Without fees,

\[
dy_b=
\frac{x_ay_br}
{x_by_a+(x_a+x_b)r}.
\]

With fees,

\[
\boxed{
dy_b=
\frac{
f^2x_ay_br
}
{
x_by_a+
(fx_b+f^2x_a)r
}
}
\]

The structure is identical.

The only difference is that the fee multiplier naturally appears in the coefficients.

---

# Looking Ahead

We'll

```

Apply the Quotient Rule

```
