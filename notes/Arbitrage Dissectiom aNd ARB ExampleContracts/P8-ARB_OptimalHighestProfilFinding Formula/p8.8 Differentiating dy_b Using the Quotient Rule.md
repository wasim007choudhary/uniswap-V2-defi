# Step 8 – Differentiating `dy_b` Using the Quotient Rule

> **Goal of this step**
>
> In **Step 7**, we simplified the arbitrage equation into a clean function that depends only on a single variable:
>
> ```
>                 xₐy_bdyₐ
> dy_b = -----------------------------------------
>         x_byₐ + (xₐ + x_b)dyₐ
> ```
>
> Now we want to determine **which value of `dyₐ` maximizes profit**.
>
> Earlier, we proved that
>
> ```
> F'(dyₐ) = dy_b' - 1
> ```
>
> and that the maximum profit occurs when
>
> ```
> F'(dyₐ) = 0
> ```
>
> Therefore,
>
> ```
> dy_b' = 1
> ```
>
> To use this condition, we must first compute the derivative of `dy_b`.

---

# Where We Are

Our arbitrage output is

```
                xₐy_bdyₐ
dy_b = -----------------------------------------
        x_byₐ + (xₐ + x_b)dyₐ
```

Notice something.

Both the numerator **and** the denominator contain

```
dyₐ
```

This means we cannot use the ordinary power rule.

Instead,

we must use the **Quotient Rule**.

---

# Recall The Quotient Rule

Suppose we have

```
       u(x)
f(x) = ------
       v(x)
```

Then its derivative is

```
         u'v - uv'
f'(x) = --------------
             v²
```

Many students memorize this as

> **Low d-high minus high d-low, over low squared.**

While this mnemonic is useful,

it's even more important to understand what each symbol represents.

---

# Identify The Numerator And Denominator

Our function is

```
                xₐy_bdyₐ
dy_b = -----------------------------------------
        x_byₐ + (xₐ + x_b)dyₐ
```

Define

```
u = xₐy_bdyₐ
```

and

```
v = x_byₐ + (xₐ + x_b)dyₐ
```

Now the Quotient Rule becomes much easier to apply.

---

# Differentiate The Numerator

The numerator is

```
u = xₐy_bdyₐ
```

Notice that

- `xₐ` is a reserve.
- `y_b` is a reserve.

For a given arbitrage opportunity,

both are constants.

Only

```
dyₐ
```

changes.

Therefore,

the numerator is simply

```
(constant) × dyₐ
```

The derivative is

```
u' = xₐy_b
```

Exactly like

```
d(5x)
-----
 dx

= 5
```

---

# Differentiate The Denominator

The denominator is

```
v = x_byₐ + (xₐ + x_b)dyₐ
```

Differentiate each term separately.

The first term

```
x_byₐ
```

contains no variable.

Its derivative is

```
0
```

The second term

```
(xₐ + x_b)dyₐ
```

is simply

```
(constant) × dyₐ
```

Therefore,

its derivative is

```
xₐ + x_b
```

Hence,

```
v' = xₐ + x_b
```

---

# Apply The Quotient Rule

Now substitute

```
u  = xₐy_bdyₐ

u' = xₐy_b

v  = x_byₐ + (xₐ + x_b)dyₐ

v' = xₐ + x_b
```

into

```
         u'v - uv'
f'(x) = --------------
             v²
```

The derivative becomes

```
             xₐy_b(x_byₐ + (xₐ + x_b)dyₐ)
             - xₐy_bdyₐ(xₐ + x_b)
dy_b' = ----------------------------------------------------------
                   (x_byₐ + (xₐ + x_b)dyₐ)²
```

Although this looks complicated,

the simplification is surprisingly easy.

---

# Expand The First Product

Expand only

```
xₐy_b(x_byₐ + (xₐ + x_b)dyₐ)
```

Result

```
xₐx_byₐy_b

+

xₐy_b(xₐ + x_b)dyₐ
```

Substitute this back.

```
xₐx_byₐy_b

+

xₐy_b(xₐ + x_b)dyₐ

-

xₐy_bdyₐ(xₐ + x_b)
```

---

# Notice The Cancellation

Look carefully at the last two terms.

```
+xₐy_b(xₐ + x_b)dyₐ
```

and

```
-xₐy_bdyₐ(xₐ + x_b)
```

These expressions are identical.

One is positive.

One is negative.

They cancel completely.

Only one term remains.

```
xₐx_byₐy_b
```

This dramatically simplifies the derivative.

---

# Final Result

After cancellation,

the derivative becomes

```
                    xₐx_byₐy_b
dy_b' = ---------------------------------------------
          (x_byₐ + (xₐ + x_b)dyₐ)²
```

This is the final derivative we needed.

---

# Why Is This Result So Nice?

Notice the numerator.

```
xₐx_byₐy_b
```

It contains **only reserve values**.

Those are constants for a given arbitrage opportunity.

The only place where

```
dyₐ
```

still appears

is inside the denominator.

This makes solving for the optimal trade size much easier.

---

# Connecting Back To The Profit Function

Earlier,

we proved

```
F'(dyₐ)

=

dy_b'

-

1
```

The maximum profit occurs when

```
F'(dyₐ)

=

0
```

Therefore,

```
dy_b'

=

1
```

We have now computed

```
dy_b'
```

so the next step is straightforward.

Simply solve

```
                    xₐx_byₐy_b
------------------------------------------------ = 1
       (x_byₐ + (xₐ + x_b)dyₐ)²
```

for the only remaining unknown,

```
dyₐ
```

That solution is the **Optimal Amount In**.

---

# ✅ Step 8 Complete

At this point we understand:

- Why the **Quotient Rule** is required.
- How to identify the numerator (`u`) and denominator (`v`).
- How to compute `u'` and `v'`.
- How to substitute them into the Quotient Rule.
- Why expanding only one product is enough.
- How two identical terms cancel perfectly.
- How the derivative simplifies to

```
                    xₐx_byₐy_b
dy_b' = ---------------------------------------------
          (x_byₐ + (xₐ + x_b)dyₐ)²
```

- Why the numerator contains only constants while the denominator contains the optimization variable `dyₐ`.
- Why this derivative is exactly what we need to solve for the **Optimal Amount In**.

In the next step, we'll set

```
dy_b' = 1
```

and solve the resulting equation algebraically to obtain the closed-form formula for the **Optimal Amount In**.
