# Step F5 – Differentiating the Fee-Adjusted Arbitrage Function

> **Goal of this step**
>
> In **Step F4**, we eliminated the intermediate variable \(dx_a\), obtaining a single expression that describes the entire arbitrage.
>
> \[
> \boxed{
> dy_b=
> \frac{
> f^2x_ay_br
> }
> {
> x_by_a+
> (fx_b+f^2x_a)r
> }
> }
> \]
>
> where
>
> \[
> f=\frac{997}{1000}.
> \]
>
> Our objective is now exactly the same as in the zero-fee derivation:
>
> **Find the value of \(r\) that maximizes profit.**

---

# Recall The Profit Function

The trader begins with

```
r
```

and finishes with

```
dy_b.
```

Therefore,

\[
\boxed{
P(r)=dy_b-r
}
\]

where

- \(P(r)\) is the arbitrage profit.
- \(r\) is the initial investment.

---

# Maximum Profit Condition

A function reaches a maximum when

\[
P'(r)=0.
\]

Differentiate the profit function.

\[
P'(r)
=
\frac{d}{dr}(dy_b-r)
\]

Since

\[
\frac{d}{dr}(r)=1,
\]

we obtain

\[
\boxed{
P'(r)=dy_b'-1.
}
\]

Setting the derivative equal to zero gives

\[
dy_b'-1=0.
\]

Therefore,

\[
\boxed{
dy_b'=1.
}
\]

This is the equation we'll eventually solve.

---

# Identify The Quotient

Our function is

\[
dy_b=
\frac{
f^2x_ay_br
}
{
x_by_a+
(fx_b+f^2x_a)r
}.
\]

To make differentiation easier,

define

\[
u=f^2x_ay_br
\]

and

\[
v=x_by_a+
(fx_b+f^2x_a)r.
\]

Then

\[
dy_b=\frac{u}{v}.
\]

---

# Differentiate The Numerator

The numerator is

\[
u=f^2x_ay_br.
\]

Everything except

\[
r
\]

is constant.

Therefore,

\[
\boxed{
u'
=
f^2x_ay_b.
}
\]

---

# Differentiate The Denominator

The denominator is

\[
v=
x_by_a+
(fx_b+f^2x_a)r.
\]

The first term

\[
x_by_a
\]

is constant.

Its derivative is

\[
0.
\]

The second term is linear in

\[
r.
\]

Therefore,

\[
\boxed{
v'
=
fx_b+f^2x_a.
}
\]

---

# Apply The Quotient Rule

Recall the Quotient Rule.

\[
\boxed{
\left(
\frac{u}{v}
\right)'
=
\frac{u'v-uv'}{v^2}
}
\]

Substitute

\[
u,\;
u',\;
v,\;
v'.
\]

This gives

\[
dy_b'
=
\frac{
f^2x_ay_b
\left(
x_by_a+
(fx_b+f^2x_a)r
\right)
-
\left(
f^2x_ay_br
\right)
\left(
fx_b+f^2x_a
\right)
}
{
\left(
x_by_a+
(fx_b+f^2x_a)r
\right)^2
}.
\]

At first glance,

this looks intimidating.

However,

almost everything cancels.

---

# Expand The First Product

Expand

\[
u'v.
\]

\[
\begin{aligned}
u'v
&=
f^2x_ay_bx_by_a
\\
&\quad+
f^2x_ay_b
(fx_b+f^2x_a)r.
\end{aligned}
\]

---

# Expand The Second Product

Now expand

\[
uv'.
\]

\[
\begin{aligned}
uv'
&=
f^2x_ay_br
(fx_b+f^2x_a).
\end{aligned}
\]

Notice something important.

The second term of

\[
u'v
\]

is

\[
f^2x_ay_b
(fx_b+f^2x_a)r,
\]

which is **identical** to

\[
uv'.
\]

Therefore,

they cancel exactly.

---

# Cancellation

After cancellation,

the numerator becomes

\[
\boxed{
f^2x_ax_by_ay_b.
}
\]

Everything containing

\[
r
\]

has disappeared.

This is one of the most elegant parts of the derivation.

The derivative simplifies dramatically.

---

# Final Derivative

Substituting the simplified numerator,

\[
\boxed{
dy_b'
=
\frac{
f^2x_ax_by_ay_b
}
{
\left(
x_by_a+
(fx_b+f^2x_a)r
\right)^2
}.
}
\]

Notice something remarkable.

The numerator now contains **only constants**.

The only variable,

\[
r,
\]

appears inside the squared denominator.

This is exactly what happened in the zero-fee derivation.

---

# Apply The Maximum Profit Condition

Earlier,

we proved

\[
dy_b'=1.
\]

Substitute the derivative.

\[
\boxed{
\frac{
f^2x_ax_by_ay_b
}
{
\left(
x_by_a+
(fx_b+f^2x_a)r
\right)^2
}
=
1.
}
\]

This equation contains only one unknown:

\[
r.
\]

Everything else is a constant determined by the pool reserves.

In the next step,

we'll solve this equation algebraically.

---

# Compare With The Zero-Fee Derivative

Without fees,

\[
dy_b'
=
\frac{
x_ax_by_ay_b
}
{
(x_by_a+(x_a+x_b)r)^2
}.
\]

With fees,

\[
\boxed{
dy_b'
=
\frac{
f^2x_ax_by_ay_b
}
{
\left(
x_by_a+
(fx_b+f^2x_a)r
\right)^2
}.
}
\]

Notice that the structure is identical.

The fee multiplier simply modifies the coefficients.

---

# Looking Ahead

Only one step remains.

We'll solve

\[
\frac{
f^2x_ax_by_ay_b
}
{
\left(
x_by_a+
(fx_b+f^2x_a)r

\]

by

dy_b'
