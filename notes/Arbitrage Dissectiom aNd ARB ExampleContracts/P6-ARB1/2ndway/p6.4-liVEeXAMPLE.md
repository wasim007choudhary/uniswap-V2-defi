# Live Example — Alex Executes a Flash Swap Arbitrage

## Scenario

Alex wants to perform an arbitrage between:

* **Router0:** Uniswap V2
* **Router1:** SushiSwap

Trading pair:

```text
DAI / WETH
```

Alex notices:

```text
Uniswap

100 DAI

↓

0.050 WETH
```

while on SushiSwap:

```text
0.050 WETH

↓

105 DAI
```

Potential profit:

```text
105 DAI

-

100 DAI

=

5 DAI
```

Alex only has:

```text
Wallet Balance

0.50 ETH

(for gas)

0 DAI
```

Even though Alex owns **no DAI**, Flash Swaps allow the arbitrage.

---

# Step 1 — Alex Opens His Wallet

Alex opens MetaMask.

```text
Wallet

↓

Alex

Address

0xAlex...
```

He connects his wallet to the arbitrage dApp.

---

# Step 2 — Alex Fills the Arbitrage Form

The frontend prepares:

```text
Pair

↓

DAI/WETH Pair
```

```text
Router0

↓

Uniswap V2
```

```text
Router1

↓

SushiSwap
```

```text
Borrow

↓

100 DAI
```

```text
Minimum Profit

↓

2 DAI
```

Internally the frontend prepares:

```solidity
SwapParams

router0 = Uniswap

router1 = Sushi

tokenIn = DAI

tokenOut = WETH

amountIn = 100 DAI

minProfit = 2 DAI
```

---

# Step 3 — Alex Clicks

```text
Execute Flash Arbitrage
```

---

# Step 4 — MetaMask Opens

MetaMask displays:

```text
Calling

flashSwap(...)
```

Alex reviews:

* Gas fee
* Network
* Contract

Then clicks:

```text
Confirm
```

*(If MetaMask is locked, Alex unlocks it using his wallet password or other configured authentication. This happens locally inside MetaMask and is unrelated to the smart contract.)*

---

# Step 5 — Transaction Broadcast

The signed transaction is sent to Ethereum.

```text
Alex

↓

Ethereum Mempool
```

Nothing has executed yet.

---

# Step 6 — Validator Mines Transaction

A validator includes the transaction in a block.

Now Solidity execution begins.

---

# Step 7 — Execution Enters

```solidity
flashSwap(...)
```

Current state:

```text
msg.sender

↓

Alex
```

---

# Step 8 — Encode Callback Data

The contract executes:

```solidity
abi.encode(
    msg.sender,
    pair,
    params
);
```

Package becomes conceptually:

```text
Alex

DAI/WETH Pair

SwapParams
```

---

# Step 9 — Call Pair.swap()

```text
Arb Contract

↓

Pair.swap(...)
```

Execution leaves the Arb Contract.

---

# Step 10 — Pair Executes

Pair performs:

* output validation
* reserve validation
* optimistic transfer

Then sends:

```text
100 DAI

↓

Arb Contract
```

The Arb Contract now owns:

```text
100 DAI
```

even though Alex never owned any DAI.

---

# Step 11 — Pair Detects Flash Swap

Pair checks:

```solidity
if (data.length > 0)
```

Since data exists:

```text
Flash Swap

↓

Callback
```

---

# Step 12 — Pair Calls Back

Execution becomes:

```text
Pair

↓

ArbContract.uniswapV2Call(...)
```

Current state:

```text
msg.sender

↓

Pair
```

Notice:

Alex is no longer `msg.sender`.

---

# Step 13 — Decode

The contract executes:

```solidity
abi.decode(...)
```

Recovered:

```text
caller

↓

Alex
```

```text
pair

↓

DAI/WETH Pair
```

```text
params

↓

SwapParams
```

Everything packed earlier has been restored.

---

# Step 14 — Execute Arbitrage

The callback calls:

```solidity
_swap(params)
```

First swap:

```text
100 DAI

↓

Uniswap

↓

0.050 WETH
```

Second swap:

```text
0.050 WETH

↓

SushiSwap

↓

105 DAI
```

Returned:

```text
amountOut

↓

105 DAI
```

---

# Step 15 — Calculate Repayment

Borrowed:

```text
100 DAI
```

Flash Swap fee:

```text
0.3009 DAI
```

Repayment:

```text
100.3009 DAI
```

---

# Step 16 — Calculate Profit

Current balance:

```text
105 DAI
```

Repay:

```text
100.3009 DAI
```

Remaining:

```text
4.6991 DAI
```

---

# Step 17 — Minimum Profit Check

Alex requested:

```text
Minimum Profit

↓

2 DAI
```

Actual profit:

```text
4.6991 DAI
```

Check:

```text
4.6991

>=

2

↓

PASS
```

If profit had been lower than 2 DAI, the transaction would revert and all state changes—including the flash loan—would be undone.

---

# Step 18 — Repay Pair

The Arb Contract transfers:

```text
100.3009 DAI

↓

Pair
```

The flash swap loan is now fully repaid.

---

# Step 19 — Send Profit

The Arb Contract transfers:

```text
4.6991 DAI

↓

Alex
```

---

# Step 20 — Callback Returns

Execution returns to:

```text
Pair.swap()
```

The Pair verifies its invariant and completes successfully.

---

# Step 21 — Transaction Finishes

Final balances:

```text
Pair

↓

Fully Repaid
```

```text
Arb Contract

↓

0 DAI
```

```text
Alex

↓

+4.6991 DAI Profit
```

Alex never supplied the 100 DAI.

The Pair temporarily supplied the capital, the arbitrage completed, the Pair was repaid within the same transaction, and Alex received the remaining profit.
